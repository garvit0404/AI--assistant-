# Future Roadmap

Planned features and architectural improvements:

- [ ] **Multi-Agent Architecture**: Separate specialized agents for coding, research, and automation.
- [ ] **Voice Assistant**: Integrated speech-to-text and text-to-speech.
- [ ] **Image Generation**: Local or API-based image generation tools.
- [ ] **Automation Workflows**: Visual builder for complex recurring tasks.
- [ ] **Meeting Management**: Calendar integration and automated summarization.
- [ ] **Local LLM Support**: Support for running reasoning engines entirely on-premise via Ollama or LocalAI.
