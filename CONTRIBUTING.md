# Contributing to AI Assistant

## Coding Standards
- Use Clean Code principles.
- Ensure all new features are documented in the `/docs` folder.
- Maintain the modular monorepo structure.

## Branching and Commits
- Use descriptive branch names: `feature/name` or `fix/issue`.
- Follow conventional commits where possible.

## Documentation Requirements
- Update the relevant `.md` file in `/docs` when changing architecture or system design.
